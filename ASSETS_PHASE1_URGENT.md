## 🎨 拍摄技巧

### **工厂/车间拍摄**
1. **黄金时间**：上午9-11点，下午2-4点（光线最佳）
2. **角度选择**：
   - 广角：展示全景和规模
   - 斜45度：增加立体感
   - 平视：真实自然
3. **构图原则**：
   - 三分法构图
   - 保持水平线平直
   - 避免杂乱元素

### **产品摄影**
1. **背景**：纯白或纯灰色背景
2. **光线**：45度侧光 + 补光（消除阴影）
3. **角度**：
   - 正面：展示全貌
   - 45度：显示立体感
   - 俯视：多产品组合
4. **对焦**：对准产品中心或标签

### **手机拍摄设置**
```
分辨率：最高分辨率模式
格式：JPG（不要用HEIC）
HDR：开启（增加细节）
网格线：开启（辅助构图）
闪光灯：关闭（用自然光）
```
