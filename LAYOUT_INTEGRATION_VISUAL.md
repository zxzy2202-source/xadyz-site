## 💻 代码示例（React/TSX）

### **标准 Layout 组件**

```tsx
// Layout.tsx
export function Layout({ children, pageTitle, pageSubtitle }) {
  return (
    <div className="min-h-screen flex flex-col">
      {/* Header */}
      <Header />

      {/* Main Content */}
      <main className="flex-1">
        {/* Page Hero */}
        <section className="bg-gradient-to-r from-blue-50 to-blue-100 py-12">
          <div className="container mx-auto px-4">
            <h1 className="text-4xl font-bold mb-2">{pageTitle}</h1>
            <p className="text-lg text-gray-600">{pageSubtitle}</p>
          </div>
        </section>

        {/* ✅ Breadcrumb - 插入这里 */}
        <div className="container mx-auto px-4 py-4">
          <Breadcrumb />
        </div>

        {/* Page Body */}
        <div className="container mx-auto px-4 py-8">
          {children}
        </div>
      </main>

      {/* Footer */}
      <Footer />
    </div>
  );
}
```

## 🎯 关键定位点

### **视觉层级**

```
1. Header (最上方，固定或滚动)
   ↓
2. PageHero (视觉焦点：大标题、图片)
   ↓
3. Breadcrumb (次要导航：小字、灰色) ← 这里
   ↓
4. PageBody (主要内容：段落、图片、表格)
   ↓
5. Footer (最下方)
```

### **用户视线流**

```
首次访问：
1. 看到 Header → 2. 看到 H1 → 3. 扫视 Breadcrumb → 4. 阅读内容

需要导航时：
1. 想回上级 → 2. 眼睛向上移动 → 3. 看到 Breadcrumb → 4. 点击
```

## ⚠️ 常见错误示例

### **❌ 错误 1: 放在 Banner 图片上**

```
┌─────────────────────────────────────┐
│  ████████████████████████████████   │
│  █ Banner Image                 █   │
│  █ 🏠 Home › Products ← 错误!   █   │ ← 会被图片遮挡
```

## ✅ 验收检查（可视化版）

### **1. 位置检查**

```
打开页面，用眼睛扫描：

1️⃣ 最上面是 Header ✅
2️⃣ 然后是 PageHero (H1) ✅
3️⃣ 然后是 Breadcrumb ✅ ← 检查这个
4️⃣ 然后是正文内容 ✅
5️⃣ 最下面是 Footer ✅

如果 Breadcrumb 不在 3️⃣ 位置，就是错的！
```

### **2. 间距检查**

```
用浏览器开发者工具测量：

┌──────────────────┐
│  H1 bottom       │
└──────────────────┘
    ↓ 8-12px ← 应该在这个范围
┌──────────────────┐
│  Breadcrumb top  │
└──────────────────┘
    ↓ 10-16px ← 应该在这个范围
┌──────────────────┐
│  Content top     │
└──────────────────┘
```

### **3. 对齐检查**

```
桌面端：
├─ Breadcrumb 左边缘应该和 H1 左边缘对齐
└─ Breadcrumb 左边缘应该和内容左边缘对齐

移动端：
├─ Breadcrumb 应该在 Container 内
└─ 有适当的左右 padding (16-24px)
```

## 📐 Figma 设计示意

```
Frame: "Page Layout"
├─ Header (Component)
├─ PageHero (Frame)
│  ├─ H1 (Text)
│  └─ Subtitle (Text)
├─ Breadcrumb (Component) ← 确保这个在独立的层
├─ PageBody (Frame)
│  └─ Content...
└─ Footer (Component)

标注：
- Breadcrumb 与 PageHero 间距: 8-12px
- Breadcrumb 与 PageBody 间距: 10-16px
- Breadcrumb 水平对齐: 与 Container 对齐
```

---

## 🎊 总结

### **记住这 5 个要点：**

1. **位置：** H1 下方、内容上方
2. **间距：** 上 8-12px、下 10-16px
3. **对齐：** 左对齐，与内容对齐
4. **层级：** 独立的层，不在 Banner 或内容内
5. **响应式：** 移动端可换行

### **一句话记忆法：**

> **"标题下面、内容上面、左边对齐、独立一层"**
