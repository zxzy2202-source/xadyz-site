## 📝 素材命名规范（强制执行）

### **标准格式**
```
{类型}-{描述}-{序号}.{扩展名}
```

### **类型前缀对照表**

| 前缀 | 类型 | 示例 |
|-----|------|------|
| `factory-` | 工厂图 | `factory-exterior-01.jpg` |
| `product-` | 产品图 | `product-thermal-paper-blank.jpg` |
| `material-` | 原料图 | `material-jumbo-rolls-warehouse.jpg` |
| `banner-` | 横幅图 | `banner-hero-home.jpg` |
| `app-` | 应用场景 | `app-retail-pos-in-use.jpg` |
| `cert-` | 证书文档 | `cert-iso9001.pdf` |

### **✅ 好的命名示例**
```
✓ factory-production-line-thermal-01.jpg
✓ product-thermal-labels-printed-sample.jpg
✓ material-ncr-jumbo-rolls-warehouse.jpg
✓ banner-hero-manufacturing-page.jpg
✓ app-logistics-warehouse-labels.jpg
✓ cert-iso9001-quality-management.pdf
```

### **❌ 错误的命名示例**
```
✗ IMG_20260203_123456.jpg        （无意义）
✗ 微信图片_202602031234.jpg      （中文）
✗ DSC_0001.JPG                   （无描述）
✗ photo.jpg                      （太简单）
✗ new_image (1) copy.jpg         （有空格和括号）
```

## 🔧 操作步骤详解

### **步骤1：准备素材文件**

#### **图片要求**
- **格式**：JPG、PNG、WebP（推荐JPG）
- **分辨率**：
  - Hero Banner：1920x1080（16:9）
  - 产品图：1200x800（3:2）
  - 缩略图：800x600（4:3）
- **文件大小**：单张 < 2MB（推荐 500KB-1MB）
- **色彩模式**：RGB（不要用CMYK）

#### **文档要求**
- **格式**：PDF（不要用DOC、DOCX）
- **文件大小**：单份 < 5MB
- **清晰度**：300 DPI以上

---

### **步骤2：批量上传素材**

#### **方法A：拖拽上传（推荐）**
```
1. 在文件管理器中选中多个文件
2. 拖拽到后台上传区域（虚线框）
3. 等待上传完成（显示绿色提示）
4. 自动刷新素材列表
```

#### **方法B：点击上传**
```
1. 点击上传区域的任意位置
2. 在文件选择对话框中选择文件
3. 支持Ctrl/Cmd多选
4. 点击"打开"开始上传
```

#### **智能分类规则**
系统会根据文件名自动分类：
- 包含 `banner` → 分类为"横幅图"
- 包含 `factory` 或 `workshop` → 分类为"工厂图"
- 包含 `product` → 分类为"产品图"
- 包含 `material` → 分类为"原料图"
- PDF/文档格式 → 分类为"文档"
- 其他 → 分类为"其他"

---

### **步骤3：审核与管理**

#### **批准素材**
```
1. 点击素材卡片上的绿色 ✓ 按钮
2. 状态变为"已批准"（绿色标签）
3. 素材可用于关联占位符
```

#### **修改分类**
```
1. 点击素材卡片上的分类下拉菜单
2. 选择正确的分类
3. 自动保存并更新
```

#### **搜索素材**
```
1. 在搜索框输入关键词
2. 支持搜索：文件名、分类、标签
3. 实时显示匹配结果
```

#### **筛选素材**
```
【按分类筛选】
- 全部类型
- 横幅图
- 工厂图
- 产品图
- 原料图
- 文档
- 其他

【按状态筛选】
- 全部状态
- 已批准（可用）
- 待审核（需处理）
- 已拒绝（质量不合格）
```

## 📊 素材质量检查清单

### **图片质量标准**

#### **✅ 合格标准**
- [ ] 清晰度高，无模糊
- [ ] 构图合理，主体突出
- [ ] 光线充足，色彩自然
- [ ] 无水印、无logo（除非有意展示）
- [ ] 分辨率达标（见上文要求）
- [ ] 文件大小适中（不要太大）
- [ ] 文件名符合规范

#### **❌ 拒绝标准**
- [ ] 模糊不清、失焦严重
- [ ] 光线太暗或过曝
- [ ] 构图混乱、主体不明
- [ ] 有明显的非相关水印
- [ ] 分辨率太低（< 800x600）
- [ ] 文件损坏或无法打开
- [ ] 内容与描述不符

## 🔗 素材与占位符关联

### **关联流程**
```
素材库（Assets）   →   占位符管理（Placeholders）   →   前端显示
     ↓                        ↓                          ↓
1. 上传并批准素材        3. 选择占位符位置             5. ImagePlaceholder
2. 记录asset_id         4. 关联asset_id               6. 显示真实图片
```

### **关联示例**
```typescript
// 占位符记录
{
  placeholder_key: 'manufacturing_hero',
  page_path: '/en/manufacturing',
  status: 'missing',        // 初始状态
  asset_id: null            // 未关联
}

// 关联素材后
{
  placeholder_key: 'manufacturing_hero',
  page_path: '/en/manufacturing',
  status: 'replaced',       // 已替换 ✅
  asset_id: 'abc-123-xyz'   // 已关联素材ID
}
```

## 🛠️ 常见问题与解决方案

### **Q1：上传失败怎么办？**
```
可能原因：
1. 文件过大（>10MB）
   → 解决：压缩图片后重试

2. 网络不稳定
   → 解决：检查网络，刷新页面重试

3. 文件格式不支持
   → 解决：转换为JPG/PNG/PDF格式

4. 存储空间不足
   → 解决：联系管理员增加Supabase存储配额
```

### **Q2：如何批量修改分类？**
```
目前方案：
1. 使用分类筛选器选出需要修改的素材
2. 逐个点击下拉菜单修改
3. 系统自动保存

未来优化：
- 添加批量选择功能
- 添加批量操作按钮
```

### **Q3：如何删除素材？**
```
当前版本：
- 不支持前端删除（防止误操作）
- 需要在Supabase后台数据库手动删除

操作步骤：
1. 登录 https://supabase.com/dashboard
2. 选择项目 → Table Editor → assets表
3. 找到对应记录 → Delete
4. 同时删除Storage中的文件
```

### **Q4：素材总数不是54个怎么办？**
```
【少于54个】
→ 按照 /ASSETS_54_CHECKLIST.md 清单补充上传

【多于54个】
→ 正常情况，可保留备用素材
→ 建议保留60-70个以便替换更新
```

### **1. 文件组织**
```
本地文件夹结构（上传前）：
/志信纸业素材/
  ├─ 01-factory/          （工厂图12张）
  ├─ 02-products/         （产品图18张）

  ├─ 05-banners/          （横幅图6张）
  └─ 06-docs/             （证书2份）
```

### **2. 批量重命名**
```bash
# 使用工具批量重命名（Windows）
# 1. 安装 PowerRename (PowerToys)
# 2. 选中文件 → 右键 → PowerRename
# 3. 搜索：IMG_  替换：factory-exterior-
# 4. 应用

# 使用脚本批量重命名（Mac/Linux）
cd /path/to/images
i=1; for f in *.jpg; do mv "$f" "factory-exterior-$(printf %02d $i).jpg"; ((i++)); done
```

### **3. 图片优化**
```
推荐工具：
- TinyPNG (https://tinypng.com/) - 在线压缩
- ImageOptim (Mac) - 批量无损压缩
- Squoosh (https://squoosh.app/) - 高级压缩

压缩目标：
- Hero Banner: 1920x1080, 500-800KB
- 产品图: 1200x800, 300-500KB
- 缩略图: 800x600, 150-300KB
```
