## 🎯 问题根源

**面包屑文字是白色的，所以在白色/浅色背景下看不见！**

从截图可以看到：
- 用户选中后显示："Home | Manufacturing | Factory Overview"
- 这些文字是**白色的**
- **被 Hero 区域的 `text-white` 样式继承了**

---

## 🔧 修复方案

**在 Breadcrumb.tsx 中使用 inline style 强制颜色，不受任何 CSS 继承影响：**

```tsx
// ✅ Before（使用 Tailwind class，可能被覆盖）
<nav className="text-sm text-gray-600">
  <a className="text-gray-600 hover:text-gray-900">

// ✅ After（使用 inline style，最高优先级）
<nav className="text-sm" style={{ color: '#4b5563' }}>
  <a style={{ color: '#4b5563' }} onMouseEnter/Leave>
```

**颜色映射：**
- `#4b5563` = gray-600（链接默认颜色）
- `#111827` = gray-900（当前页 + hover 颜色）
- `#9ca3af` = gray-400（分隔符颜色）
