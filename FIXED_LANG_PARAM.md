## 🎯 问题原因

**ProductsPage 缺少关键参数！**

```tsx
// ❌ Before（问题）
<PageShell hero={heroContent}>

// ✅ After（修复）
<PageShell lang={lang} hero={heroContent}>
```

**没有传 `lang` 参数，导致 Breadcrumb 无法正确获取语言！**
