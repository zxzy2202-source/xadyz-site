# 志信纸业 (Zhixin Paper) 网站页面地图

> 支持语言：`/en/` 英语 | `/ru/` 俄语 | `/zh/` 中文

---

## 一、前台页面（公开访问）

### 1. 首页
| 路径 | 说明 |
|------|------|
| `/` | 重定向至 `/en/` |
| `/{lang}/` | 首页（en/ru/zh） |

---

### 2. 产品中心 (Products)

| 路径 | 页面 | 说明 |
|------|------|------|
| `/{lang}/products` | 产品总览 | ProductsPage |
| `/{lang}/thermal-paper-rolls` | 热敏纸卷 | ThermalPaperPage |
| `/{lang}/thermal-paper-rolls/blank` | 空白热敏纸卷 | |
| `/{lang}/thermal-paper-rolls/printed` | 印刷热敏纸卷 | |
| `/{lang}/thermal-labels` | 热敏标签 | ThermalLabelsPage |
| `/{lang}/thermal-labels/blank` | 空白热敏标签 | |
| `/{lang}/thermal-labels/printed` | 印刷热敏标签 | |
| `/{lang}/ncr-forms` | NCR 表格 | NCRFormsPage |
| `/{lang}/ncr-forms/blank` | 空白 NCR 表格 | |
| `/{lang}/ncr-forms/printed` | 印刷 NCR 表格 | |
| `/{lang}/ncr-forms/continuous` | 连续式 NCR 表格 | |

### 3. 材料供应 (Material Supply)

| 路径 | 页面 | 说明 |
|------|------|------|
| `/{lang}/material-supply` | 材料供应总览 | MaterialSupplyOverviewPage |
| `/{lang}/material-supply/thermal-jumbo-rolls` | 热敏 Jumbo 大卷 | JumboRollsPage |
| `/{lang}/material-supply/self-adhesive-jumbo-rolls` | 不干胶卷材 | SelfAdhesiveJumboPage |
| `/{lang}/material-supply/self-adhesive-sheets` | 不干胶平张 | SelfAdhesiveSheetsPage |
| `/{lang}/material-supply/ncr-jumbo-rolls` | NCR 卷材 | NCRJumboPage |
| `/{lang}/material-supply/ncr-sheets` | NCR 平张 | NCRSheetsPage |

**重定向：**
- `/material-supply/self-adhesive` → `self-adhesive-jumbo-rolls`
- `/material-supply/ncr` → `ncr-jumbo-rolls`
- `/material-supply/thermal-jumbo` → `thermal-jumbo-rolls`
