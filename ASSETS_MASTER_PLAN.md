## 🎨 拍摄技巧速查

### **工厂拍摄**
```
✓ 时间：上午9-11点，下午2-4点
✓ 角度：广角全景 + 斜45度
✓ 构图：三分法，水平线平直
✓ 光线：明亮、自然光优先
✓ 整理：清理杂物，整洁有序
```

### **产品拍摄**
```
✓ 背景：纯白或纯灰
✓ 布光：45度侧光 + 补光
✓ 角度：正面 + 45度 + 俯视
✓ 对焦：产品中心或标签
✓ 多拍：每个角度拍5-10张
```

### **应用场景**
```
✓ 真实：真实使用环境
✓ 自然：抓拍，不要摆拍
✓ 细节：标签内容清晰
✓ 故事：有场景感和代入感
✓ 许可：提前获得拍摄许可
```

## 📤 上传操作流程

### **准备工作**
1. 按照阶段整理文件夹
2. 统一重命名文件
3. 压缩优化图片（每张<1MB）
4. 检查文件格式（JPG/PDF）

### **批量上传**
1. 登录：https://xadyz.com/admin/assets
2. 拖拽文件到上传区
3. 等待上传完成（绿色提示）
4. 检查总数是否正确

### **审核管理**
1. 查看自动分类是否正确
2. 修改错误的分类
3. 点击✓批准所有合格素材
4. 拒绝❌模糊或质量差的素材
