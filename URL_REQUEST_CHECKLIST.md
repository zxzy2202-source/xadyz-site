## 💡 使用技巧

### **方法1: 逐个请求（推荐）**

1. 打开 Google Search Console
2. 点击顶部搜索框 "检查任意网址"
3. 复制粘贴一个URL
4. 点击"测试实时网址"
5. 等待测试完成（5-30秒）
6. 如果显示"网址可编入 Google 索引" → 点击"请求编入索引"
7. 重复下一个URL

### **方法2: 使用检查工具**

GSC → 左侧菜单 → URL检查 → 输入URL → 请求索引

### **方法3: 批量验证已索引状态**

在Google搜索框输入：
```
site:xadyz.com/en/products
```

有结果 = 已索引，无需请求  
无结果 = 未索引，需要请求
