## 🎨 高级拍摄技巧

### **产品细节微距摄影**
```
设备：
- 微距镜头或手机微距模式
- 三脚架（防抖）
- 柔光箱或柔光板

布光：
- 主光源：45度侧光
- 补光：反光板填充阴影
- 背光：勾勒边缘（可选）

拍摄参数（手机）：
- 专业模式
- ISO 100-200（低感光降噪）
- 快门速度：1/125s以上
- 白平衡：日光或自定义
```

### **工厂环境摄影**
```
广角拍摄：
- 使用超广角镜头（16-24mm等效）
- 手机全景模式
- 注意畸变校正

HDR技巧：
- 开启HDR模式
- 处理明暗对比大的场景
- 保留高光和暗部细节

后期处理：
- 调整曝光和对比度
- 校正倾斜（保持水平）
- 裁剪到标准比例
- 适度锐化（增强细节）
```
