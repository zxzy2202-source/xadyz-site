# 🏷 投标/项目客户询盘分级规则（A/B/C）

> **适用于：Government & Tenders页面 + Tender Pack表单**  
> **目标：第一时间判断值不值得投入资源**

---

## 📊 一、分级总原则（先给结论）

**不是"谁先问"，而是：**

- 是否已中标 / 接近中标
- 是否有明确项目
- 是否匹配你"成品制造能力"

---

## 🎯 二、A / B / C 客户快速判断表（核心）

| 级别 | 一句话判断 | 是否重点 | 响应时间 | 投入资源 |
|------|----------|---------|---------|---------|
| **A级（黄金）** | 已中标 / 明确项目，找合格供应商 | ⭐⭐⭐⭐⭐ | **24小时内** | **项目负责人** |
| **B级（潜力）** | 在准备投标，技术文件阶段 | ⭐⭐⭐ | **48小时内** | **销售团队** |
| **C级（观察）** | 泛泛询问 / 信息收集 | ⭐ | **72小时内** | **模板回复** |

---

## 🥇 三、A级投标客户（必须重点跟）

### ✅ 判定条件（满足 ≥2 条即可）

- [ ] 明确说明 **"已中标 / 我们是中标方"**
- [ ] 提供 **招标编号 / 项目名称**
- [ ] 需要 **Tender Pack / 技术文件**
- [ ] 项目数量 / 周期明确
- [ ] 产品 = Thermal Paper Rolls / Thermal Labels / NCR Forms（含Continuous）

### 🔥 特征关键词（表单/邮件中常见）

**English:**
- "tender winner"
- "project awarded"
- "government contract"
- "technical documentation required"
- "continuous supply"
- "we won the bid"
- "supplier qualification needed"

**Russian:**
- "победитель тендера"
- "контракт получен"
- "государственный контракт"
- "техническая документация требуется"
- "долгосрочные поставки"

### 🎯 跟进策略（A级）

**时间线：**
- ⏱ **24小时内首次回复**
- 📞 **48小时内电话/视频沟通**
- 📄 **72小时内提供完整文件包**

**负责人：**
- 👤 指定 **项目负责人（不是普通销售）**
- 👥 必要时组建 **项目支持小组**

**提供内容：**
1. **产品规格说明**（详细版）
2. **质量/产能说明**（工厂照片+数据）
3. **项目支持说明**（交付能力证明）
4. **合规文件**（ISO证书/检测报告等）

**沟通策略：**
- ❌ 不先谈价格
- ✅ 先谈 **合规 / 稳定 / 交付能力**
- ✅ 强调 **批次一致性**
- ✅ 提供 **项目参考（匿名化）**

**内部协作：**
- 🔔 立即通知项目经理
- 📊 CRM标记为"Hot Lead"
- 📅 设置每日跟进提醒

- [ ] 询问供应商资质

### 🧭 特征关键词

- "technical requirements"
- "need supplier info"

### 🎯 跟进策略（B级）

**时间线：**
- ⏱ **48小时内首次回复**
- 📧 **7天内发送详细资料**

**提供内容：**
1. **产品能力概述**（标准版）
2. **标准规格范围**
3. **项目经验**（不点名）

**沟通策略：**

- ✅ 了解竞争对手情况
- ✅ 提供差异化优势

**内部协作：**
- 📅 设置 **7-14天跟进提醒**
- 📊 CRM标记为"Warm Lead"
- 🔄 定期更新项目进展

**转化路径：**
```
B级客户 → 提供优质支持 → 进入投标供应商名单 → 
等待项目确认 → 转化为A级
```

## 🥉 五、C级投标客户（过滤/观察）

### ❌ 常见情况

- [ ] 不清楚项目
- [ ] 没有项目编号
- [ ] 问价为主
- [ ] 想要"完整投标文件模板"
- [ ] 产品与主营不匹配（要材料不要成品）
- [ ] 邮箱是免费邮箱（gmail/yahoo/qq）

### 🧾 特征关键词

**English:**
- "just checking"
- "for reference"
- "price list"
- "sample only"
- "general inquiry"
- "what's your best price"

**Russian:**
- "просто узнаю"
- "для справки"
- "прайс-лист"
- "только образцы"
- "какая цена"

### 🎯 跟进策略（C级）

**时间线：**
- ⏱ **72小时内模板回复**
- 📧 使用 **自动回复模板**

**提供内容：**
- 公司能力简介（1页PDF）
- 产品范围说明
- 联系方式

**沟通策略：**
- ❌ 不投入项目资源
- ❌ 不定制文件
- ❌ 不安排视频会议
- ✅ 标记为 **观察**
- ✅ 加入邮件列表（月度更新）

**内部协作：**
- 📊 CRM标记为"Cold Lead"
- 🔕 不设置提醒
- 📁 归档到"低优先级"

**例外情况：**
如果C级客户后续提供项目信息，立即重新评级！

## 📝 六、表单字段 → 自动分级逻辑（强烈建议）

### Tender Pack表单中，以下字段**直接决定分级**

#### **1️⃣ Project Type（项目类型）**
```
- Government Tender ✅
- Enterprise Project ✅
- Long-term Contract ✅
- General Inquiry ❌
```

#### **2️⃣ Project Status（项目状态）**
```
- Awarded (已中标) → A级
- Bidding (投标中) → B级
- Planning (计划中) → B级/C级
- Inquiry (询问) → C级
```

#### **3️⃣ Products of Interest（产品需求）**
```
成品（优先）：
- Thermal Paper Rolls ✅
- Thermal Labels ✅
- NCR Forms ✅

材料（降级）：
- Thermal Jumbo Rolls → 转Material Supply
- Self-Adhesive Materials → 转Material Supply
```

#### **4️⃣ Project Details（项目详情）**
```
有以下信息 → 优先级提升：
- Tender Number (招标编号) ⭐⭐⭐
- Project Name (项目名称) ⭐⭐
- Volume/Timeline (数量/时间) ⭐⭐
- Documentation Required (需要文档) ⭐
```

### 🤖 自动分级规则示例

```javascript
// 自动分级逻辑
function calculateLeadGrade(formData) {
  let score = 0;
  
  // Project Status权重最高
  if (formData.projectStatus === 'Awarded') score += 100;
  if (formData.projectStatus === 'Bidding') score += 60;
  if (formData.projectStatus === 'Planning') score += 30;
  
  // 有招标编号+50分
  if (formData.tenderNumber) score += 50;
  
  // 产品匹配度
  if (formData.products.includes('NCR Forms')) score += 20;
  if (formData.products.includes('Thermal Paper Rolls')) score += 15;
  
  // 项目详情完整度
  if (formData.volume) score += 10;
  if (formData.timeline) score += 10;
  
  // 分级判断
  if (score >= 100) return 'A';
  if (score >= 60) return 'B';
  return 'C';
}
```

## ⚠️ 十一、特殊情况处理

### 1. 大客户（无论ABC都要重视）

**判断标准：**
- 年采购量 > 100吨
- 政府长期合同
- 知名企业集团

**处理方式：**
- 直接升级为A级
- 老板/高管直接对接

---

### 2. 老客户回流

**判断标准：**
- 曾经合作过的客户
- 历史订单记录

**处理方式：**
- 直接升级为B级或A级
- 查看历史合作记录
- 安排原销售对接

---

### 3. 转介绍客户

**判断标准：**
- 现有客户推荐
- 合作伙伴介绍

**处理方式：**
- 至少B级处理
- 感谢推荐人
- 优先安排资源

---

## 🔄 十二、分级动态调整

**C级 → B级：**
- 提供项目信息
- 明确时间线
- 需要技术支持

**B级 → A级：**
- 确认中标
- 项目启动
- 需要供应商资质

**降级触发：**
- 长期无响应（>30天）
- 项目取消/延期
- 需求不匹配
